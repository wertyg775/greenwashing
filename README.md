# Greenwashing Evaluation Framework
This project is an AI-powered analyzer designed to detect greenwashing in corporate claims.

## Deployment Instructions
The application is automatically deployed to Hugging Face Spaces via GitHub Actions.

## Local Development
To run this project on your machine, ensure you have [Docker](https://docs.docker.com/get-docker/) and [Docker Compose](https://docs.docker.com/compose/install/) installed.

### Run with Docker Compose
You can build the image and start the application with a single command:

```bash
docker-compose up --build
